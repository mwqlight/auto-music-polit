# 核心功能开发和完善
1. 当前项目的核心音乐源爬虫功能，需要能够分析本地的视频文件，提取出其中的音乐内容，还需要能够支持分析视频链接，提取出视频中的音乐内容。还需要支持音乐搜索，就是用户提供一个音乐名称，然后去全网开源的地址去爬取，这些来源的音乐被默认放入到音乐列表中，并且默认分组为系统，这个功能实现后，需要用一个真实的视频链接进行测试。
2. 音乐列表功能就是就是将爬虫获取的音乐和本地上传的音乐的合集，需要能展示出来进行列表播放，以及随机播放，还是就是搜索。
3. 我的收藏和音乐列表比较类似，就是将音乐列表中，用户比较喜欢的音乐，收藏起来，然后可以在我的收藏中进行播放。
4. 音乐分组功能，就是用户可以将音乐列表中的音乐，按照不同的分类进行分组，比如用户可以将所有的系统音乐分组为系统，将所有的用户上传的音乐分组为用户，将所有的收藏音乐分组为收藏。这个功能在音乐列表和我的收藏里体现。
5. 音乐实验室功能，可以听声识曲，还有就是音频编辑功能，比如用户可以将音乐列表中的音乐，进行剪切，合并，添加音效等操作。
6. 注意UI风格是需要为多元的混合风，摇滚，恬静，田园，高科技风等，需要根据当前的歌曲播放的曲种来动态切换，默认就是高科技风。
